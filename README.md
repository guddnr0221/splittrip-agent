# SplitTrip — 여행 경비 정산 도우미

여러 영수증의 품목·수량·금액을 직접 입력하고, 참여자별 구매 수량을 배분해 최종 송금액을 계산하는 브라우저 기반 웹앱입니다.

공개 실행 URL: https://guddnr0221.github.io/splittrip-agent/

## 주요 기능

- 참여자 2~8명 등록
- 여러 영수증 및 결제자 관리
- 원·엔·달러·유로·파운드·위안 통화 설정
- 수량·개당 가격·품목 총액의 양방향 자동 계산
- 품목 구매 수량을 넘지 않는 인원별 배분
- 미입력 및 미배분 내역 검증
- 영수증별 합계와 통화별 전체 합계
- 각 참여자가 결제자에게 보낼 최종 금액 계산
- 브라우저 자동 저장

## 실행

별도 빌드나 API 키 없이 정적 파일만으로 동작합니다.

```bash
python -m http.server 4173
```

브라우저에서 `http://localhost:4173`으로 접속합니다.
