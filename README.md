# SplitTrip — AI 여행 경비 정산 Agent

영수증 이미지를 OCR로 읽고, 품목 수량을 참여자별로 배분해 각자의 송금액을 계산하는 브라우저 기반 웹앱입니다. OCR 처리는 브라우저 안에서 이루어지며 사진을 별도 서버로 전송하지 않습니다.

공개 실행 URL: https://guddnr0221.github.io/splittrip-agent/

## 실행

별도 빌드 없이 정적 파일만으로 동작합니다. 프로젝트 폴더에서 아래 명령을 실행하세요.

```bash
python -m http.server 4173
```

브라우저에서 `http://localhost:4173`으로 접속합니다.

## 주요 기능

- 참여자 2~8명 등록
- 영수증 이미지 여러 장 업로드
- 앱에 포함된 Tesseract.js 기반 한국어·영어·일본어 OCR
- OCR 결과 품목명·수량·단가 수정
- 원/엔/달러/유로/파운드/위안 자동 감지 및 수동 변경
- 품목별 수량을 초과하지 않는 참여자 배분
- 미배분 품목 검증 및 안내
- 여러 영수증의 개인별 송금액 합산
- 브라우저 자동 저장

## 공개 URL 배포

이 폴더 전체를 GitHub 저장소의 최상위에 올린 뒤 `Settings → Pages → Deploy from a branch → main / (root)`를 선택하면 됩니다. 자세한 내용은 `DEPLOYMENT.md`를 참고하세요.

OCR 엔진과 언어 파일이 `vendor/`에 포함되어 있어 별도 API 키가 필요하지 않습니다. 첫 OCR 실행은 기기 성능에 따라 수 초 정도 걸릴 수 있습니다.
